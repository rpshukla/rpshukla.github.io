napkin

Napkin is little programming language/environment for doing quick "back-of-the-napkin" math. My goal is for it to be a lightweight alternative to things like MATLAB and NumPy. It has native support for complex numbers and will eventually support matrix arithmetic.

I am writing the napkin interpreter from scratch using only standard C++ libraries. This includes a hand-written recursive descent parser.
